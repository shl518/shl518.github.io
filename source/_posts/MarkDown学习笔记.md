### 写入代码

写 ***函数名称*** 用反引号将其包起来 `printf()`函数

写 ***函数块*** 用三个反引号将其包起来并指定一种语言

```C
while(all_items[c].a!=0)
    {
        int a0=all_items[c].a,p0=all_items[c].p;
        for(t=c+1;all_items[t].p==all_items[c].p;t++)
        {
            a0+=all_items[t].a;
        }
        printf("%d %d ",a0,p0);
        c=t;
    }
```
